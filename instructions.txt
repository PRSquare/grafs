	Для начала использования программы импортируйте файл
или воспользуйтесь соответствующими пунктами меню для
создания вершин и рёбер. 
	Открыть файл можно нажав на пункт меню File 
в левом верхнем углу экрана. Далее необходимо выбрать 
тип импортируемого файла: Матрица смежности, матрица 
инцендентности или файл года графа.
	Для изменения положения вершин и удаления вершин/рёбер
воспользуйтесь инструментом Select vertex or edge.
При нажатии ЛКМ на вершину или ребро оно станет активным
(Цвет станет красным). Чтобы отменить выделение нажмите
ПКМ. При выбраной вершине повторное нажатие ЛКМ изменит
Положение этой вершины. При выбраной вершине или выбраном
ребре нажате клавиши Delete удалит эту вершину/ребро.
	Для создания новых вершин используйте инструмент Create
vertex. Нажатие ЛКМ создвёт новую вершину. 
	Для создания рёбер используйте инструмент Create edge. 
Нажатие ЛКМ по вершинам делает их активными (Зелёный цвет). 
Нажатие ПКМ отменяет выделение. Когда выбраны две вершины 
они соединяются ребром. 
	Исполльзуйте кнопку Undo для отмены
последних действий. 
	В правой стороне окна расположена
панель свойств объекта. С помощю неё можно менять 
параметры вершин/рёбер.
